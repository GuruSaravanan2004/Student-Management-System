# 🎓 Student Management System

A Java-based Student Management System built using JDBC and MySQL.

---

## 🚀 Features

- Add, View, Update Students
- Course Management
- Student Enrollment (Many-to-Many)
- Grade Assignment
- GPA Calculation
- Login System (Admin & Student)
- Role-Based Access Control

---

## 🛠️ Tech Stack

- Java (JDK 17+)
- MySQL
- JDBC
- IntelliJ IDEA

---

## 🧠 Concepts Used

- OOP (Encapsulation, Abstraction)
- JDBC (Connection, PreparedStatement, ResultSet)
- SQL (CRUD, JOIN, Constraints)
- Many-to-Many Relationship
- Role-Based Authentication

---

## 📂 Project Structure
src/
├── dao/
├── model/
├── utils/
└── Main.java

---

## 🧪 How to Run

1. Install MySQL and create database `sms`
2. Run SQL scripts to create tables
3. Configure DBConnection.java
4. Run Main.java
